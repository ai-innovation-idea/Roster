# Project Documentation

## Project Structure
- `src/` - Source files for the Spring Boot application
- `resources/` - Configuration and resource files
- `test/` - Unit and integration test files

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/ai-innovation-idea/Roster.git
   cd Roster
   ```
2. Ensure you have Java 11+ and Maven installed.

## Build and Run Commands
- To build the project, use:
   ```bash
   mvn clean install
   ```
- To run the application, use:
   ```bash
   mvn spring-boot:run
   ```

## Next Steps for Spring Boot Development
- Explore the main application class in `src/main/java/com/example/RosterApplication.java`.
- Implement new features based on the requirements.
- Write tests for your new features using JUnit.
